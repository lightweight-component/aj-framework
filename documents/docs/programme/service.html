Web 项目运行起来有其一套的“虚拟目录”。下面组件能提供这些目录（也就是 url），其中最为直观的自然是“一个文件夹一个目录”，目录里面包含若干 html/jsp 这样子。另外我们还可以不用磁盘文件/文件夹定义虚拟目录，各种 servlet 也行，通过 web.xml 或者注释定义，这种方式不容易看一目了然（但是 Eclipse 也为我们准备了可以查阅的地方）。

通用 Web 模块

提供基础 Web 服务，可选择使用

  • /util_service/htmleditor_iframe：可设置 basePath 的 iframe 页面，参数：basePath
  • /util_service/captchaImg：简易图片验证码
  • /util_service/http_proxy：HTTP 代理
  • /util_service/ip_location：根据请求 ip 显示所在区域
  • /util_service/weather：如果没有指定城市，则默认依据 ip 地址来得到城市（TODO）
  • /util_service/remoteConsole：方便手机 web 开发调试输出调试信息
  • /util_service/under_construction：显示“网站正在建设中的”图片
  • /util_service/sendMail：通过 Socket 发送邮件

接口服务

接口不限定特定的客户端或服务端，使得 Android/iOS 客户端/HTML5和后台 WebService 数据操作均可适用。接口遵循以下原则。

  • HTTP 协议通信和 JSON 数据格式，其响应 ContentType 为“application/json”
  • URL 面向资源设计，即采用 HTTP RESTful 之风格
  • 字符使用 UTF-8 编码
