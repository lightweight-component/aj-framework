AJAXJS 推荐使用 HTML 页面或 JSP。其中 JSP 之中不鼓励使用 <%...%> Java 语句，而是鼓励使用 JSTL 或自定义标签库，另外还有若干基于 Tag Files 的自定义标签。無論表示层怎么选型，最终落实的是HTML5+CSS3+JS。

前端库说明

从 AJAXJS 的字面上看，ajax.js 很像一个 JavaScript 库，它怎么变成 Java 后台框架呢？的确没错，从开发的顺序上说，它首先是个前端库，包含不少 UI 组件，PC端、手机端的也有；其次再是个 Java 后台，我把 AJAXJS 这个名字继续使用了。无论说法如何，都是笔者业余时间“原创”的作品。他既包含前端库，也提供完整的后端，为一全栈方案。两者可独立使用，彼此可游离出去。若前端代码亦能助你一臂之力，那则是善莫大焉。
