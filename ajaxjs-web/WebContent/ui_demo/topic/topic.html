概述

为针对现有模板组件的不足，特作出第二版的改进。

任意一页面，都必须插入以下代码。

<!DOCTYPE html>
<html>
	<head>
		<!--请不要修改此处文件 Start-->
		<meta content="text/html; charset=utf-8" http-equiv="content-type" />
		<meta name="viewport" content="width=320,user-scalable=0,initial-scale=1.0,maximum-scale=1.0, minimum-scale=1.0" />

		<!-- 样式文件 -->
		<link rel="stylesheet" type="text/css" href="../common/v2/common.css" />
		
		<script src="../common/v2/bigfoot/js/lang.js"></script>
		<script src="../common/v2/bigfoot/js/dom.js"></script>
		<script src="../common/v2/bigfoot/js/widget/list.js"></script>
		<script src="../common/v2/bigfoot/js/widget/tab.js"></script>
		<!--// 请不要修改此处文件 End-->
		
		<!-- 设置页面标题 -->
		<title>“东方之星”翻沉事故</title>
	</head>
	<body>
		<div class="body">
			<!--这里放置专题的内容-->
		</div>
	</body>
	<!--请不要修改此处文件 Start-->
	<script src="../common/v2/bigfoot/js/topic.js"></script>
	<!--// 请不要修改此处文件 End-->
</html>

列表组件

代码：<ul class="list col_0" sid="2268" pagesize="10"></ul>

属性析义：

  1. class 中的前半段 list 必须保留，中间以空格分隔，后半段为列表类型，必填，有如下值：
    • col_0 : 纯文字列表
    • col_1 : 单列图文列表
    • col_2 : 双列图文列表
    • col_3 : 三列图文列表
  2. sid 栏目id，数字，必填
  3. pagesize 读取记录笔数，数字，不填默认 5 笔
  4. pager 是否需要分页，
    • true，分页
    • false，不分页（默认）
  5. is_append分页新增的内容是“覆盖”还是“追加”
    • true，覆盖
    • false，追加（默认），适合手机端方式

纯文字列表例子

代码：

<ul class="list col_0" sid="2268"></ul>

    单列图文例子,分页并指定读取分页数

    代码：

    <ul class="list col_1" sid="2268" pagesize="3" pager="true"></ul>

    双列图文例子，分页，覆盖方式

    代码：

    <ul class="list col_2" sid="2268" pager="true"></ul>

    三列图文例子，分页，追加方式

    代码：

    <ul class="list col_2" sid="2268" pager="true" is_append="true"></ul>

    Tab 组件

    Tab 组件为多标签页组件，可承载丰富的内容，却不占额外的面积。一个完整的 Tab 可设定的有 Tab 标题和 Tab 内容区域。设定的内容可以是任意 HTML，也可以内嵌列表组件或相册。

    代码：

    <!-- Tab 组件 -->
    <div class="staticTab tab2 center">
          <ul>
            <li>公司简介</li>
            <li>资质证照</li>
            <li>荣誉证书</li>
          </ul>
          <div class="content">
            <div>
              自古以来，……
            </div>
            <div>
              之所以如此，端在于中国传统中存在着发达的契约。……
            </div>
            <div>
              笔者出于个人兴趣，在关注这些已经整理出版的、卷帙浩繁的契约文献的同时，也游走各地，……
            </div>
          </div>
        </div>
