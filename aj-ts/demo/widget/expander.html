展开闭合器 expander

常用于展开正文详情。提供过渡效果的图层，如果不需要把 div class="mask" 去掉即可。注意加入容器的 padding 会导致关闭时高度异常。

属性 含义 类型 是否必填，默认值
openHeight 展开状态的高度 Number n, 200
closeHeight 闭合状态的高度 Number n, 50
<slot> 正文内容扩展 String n

例子：

<aj-expander :close-height="50">……</aj-expander>
