“大脚 Bigfoot”是一款基于 Java 平台原创的 Web 框架，继承了 Java 平台的高效、安全、稳定、跨平台等诸多优势，但却摒弃了传统 SSH 企业级架构所带来的庞大和臃肿，汲取了 PHP 的简洁和方便，非常适合互联网中小型网站的应用。
  • 免费、开源、简单、概念清晰；力求高聚合、低耦合
  • 一体化全栈式方案，前端库可以游离于服务框架，反之亦然
  • 满足业界技术点最低消费：MVC+Ioc+AOP+ORM+JSON，轻量级
  • 提供大量的接口适合二次扩展
  • 业务逻辑基于 Rhino 规则引擎
  • 混合语言编程，提供服务端 JS 环境，特别适合从前端过来的开发人员
  • 完备的文档、充足的注释，具备 JUnit 单元测试
  • 谨慎采用第三方库，当前基本免 JAR 依赖（除数据库驱动外）。提倡“够用就好、就地取材”

  • 后端基础组件：
    SQLite/MySQL 数据库、
    数据连接池采用 Tomcat Pool、
    使用 TagFiles/SimpleTag 页面标签、
    日志采用 JDK 自带的 Logger、
    JSON 转换让 JDK 自带的 Rhino 转换
    DOM 解析 XML、Socket 发送邮件

  • 前端组件 Bigfoot.js：
    HTML5 + CSS3 + 原生 JavaScript
    基于 Less.js 预编译、Nodejs 构建系统
    响应式PC 端和手机端，支持 IE8+/FF/Chrome/Android 浏览器/iOS Safari，兼容 Ext JS 富客户端
    演示： tab lightbox
